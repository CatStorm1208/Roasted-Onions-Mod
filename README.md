# RoastedOnions
This Forge-Mod adds delicious (mostly) Danish cuisine into the game.\
Game version: 1.18.2
Forge version: 40.1.80 or above

## Dependencies
### Farmers Delight
Version 1.1.2 or above. Mandatory.
### Oh the biomes you'll go
Version 1.4 or above. Not Mandatory.
