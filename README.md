# RoastedOnions
This Forge-Mod adds delicious Danish cuisine into the game.\
Game version: 1.18.2
Forge version: 40.1.80 or above

## Dependencies
- Farmersdelight 1.1.2 or above 
- Oh the biomes you'll go 1.4 or above
